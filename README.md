🎟️ Nikhut Ticketing

🔗 Live Site: https://nikhut-ticketing.vercel.app
🔗 Netlify Deployment: https://nikhutticketing.netlify.app

A modern and responsive online ticket booking platform designed for smooth seat selection and ticket purchasing experience. The project focuses on interactive UI, responsive layouts, and user-friendly booking flow.

🚀 Features
🎫 Online ticket booking interface
🪑 Interactive seat selection system
💳 Ticket purchase workflow
✅ Booking success confirmation page
📱 Fully responsive design
⚡ Smooth user experience with dynamic interactions
🎨 Clean and modern UI design

🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (Vanilla JS)

📁 Project Structure
├── images/               # Image assets
├── scripts/              # JavaScript files
├── index.html            # Main booking page
├── Success.html          # Booking success page
├── tailwind.config.js    # Tailwind configuration
├── README.md             # Project documentation
└── .gitattributes


⚙️ Setup & Usage
Clone the repository:
git clone https://github.com/istewakhassantewak/Nikhut-Ticketing.git
Navigate to the project folder:
cd Nikhut-Ticketing
Open index.html in your browser.
